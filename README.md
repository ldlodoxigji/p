# Retail Insights Demo

Набор статичных страниц для демонстрации того, как можно представить результаты парсинга: сводные KPI, таблицы и интерактивные графики на Bootstrap, Chart.js и Plotly.js.

## Как запустить
1. Откройте `public/index.html` напрямую в браузере или поднимите статический сервер из корня репозитория:
   ```bash
   npx serve public
   ```
2. Переключайтесь между `index.html` (обзор) и `visualizations.html` (графики) через навигацию вверху.

## Как взаимодействовать
- Наводите курсор или нажимайте на элементы графиков, чтобы видеть всплывающие подсказки.
- Используйте легенду Chart.js/Plotly, чтобы включать или скрывать отдельные серии.
- Масштабируйте графики Plotly колёсиком мыши или жестами на трекпаде.
- Сжимайте или расширяйте окно: карточки и графики автоматически адаптируются под ширину экрана.

## Как поменять данные
- Тестовые наборы лежат в `public/js/data.js`. Обновите массивы или подставьте свои значения — визуализации используют их сразу после перезагрузки страницы.

## Системные зависимости
- Node.js не требуется для просмотра статичных страниц, но пригодится, если захотите поднимать локальный сервер через `npx serve`.
